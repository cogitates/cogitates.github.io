Deepin安装索引项

Diposting di 2019-07-12 | Edited on 2019-07-17

因为之前给Deepin的硬盘分区是60G，随着deepin成为自己的主力系统。这个空间不是很够用。有重装打算，重装肯定要做好备份。因此在这简单总结一下，如果后续重装系统的话，这篇文章也是一个索引。

格掉系统之前应该做什么备份?

  1. .vimrc
  2. .vscode
  3. .vscode-cpptools
  4. home分区中的code,mynote,myblog文件夹
  5. u1807a

重装系统后要安装什么?

  1. chrome(官网)
  2. chroium(deepin软件商店) + 谷歌访问助手
  3. vscode
  4. jetbrain toolboxs
  5. intellij IDEA
  6. mysql + mysql workbench
  7. qq
  8. wechat
  9. spotify
  10. 显卡驱动更改为大黄蜂
  11. /etc/default/grub 更改CPU调度设置 CPU调度
  12. 博客需要的东西 nodejs npm next主题7.1.4 blog恢复参考
  13. oh-my-zsh
  14. pycharm
  15. typora
  16. VLC视频播放器
  17. stacer 磁盘清理
  18. git
  19. python3 + python2 (numpy tkinter tensorflow )
  20. jdk11 install
  21. 福昕阅读器
  22. electron-ssr
  23. netease-music
