Linux 显卡问题

Posted on 2019-07-26

安装Linux时，遇到的很多问题都和显卡有问题，下面记录一些和显卡相关的参数。

  1. nomodeset 不载入所有关于显卡的驱动
  2. nouveau.modeset=0 关闭NVIDIA显卡的驱动，反之，=1为开启
  3. i915.modeset=0 关闭Intel显卡的驱动
  4. xforcevesa 或 radeon.modeset=0 xforcevesa跟AMD显卡有关的设定
  5. acpi=off 电源相关设定，OS无法处理，交给BIOS处理
